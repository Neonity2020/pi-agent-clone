# Raw Data

此目录用于存放非 AI 生成的原始数据 (Raw Data)。

## 目录用途

- 供人类手动添加的原始文件
- 数据集、配置文件、模板等
- AI 生成代码或文档的参考素材
- 测试数据、样本文件

## 与 .agent-output 的区别

| 目录 | 内容 | 生成方式 |
|------|------|----------|
| `.agent-output/` | AI 生成的代码、文档、日志 | 自动生成 |
| `raw-data/` | 人类添加的原始数据 | 手动添加 |

## 建议的子目录结构

```
raw-data/
├── datasets/          # 数据集
├── templates/         # 代码/文档模板
├── config/            # 配置文件
├── samples/           # 样本文件
└── reference/         # 参考资料
```

## 注意事项

- 此目录中的内容**会被提交到 Git**
- 请勿在此目录存放敏感信息（如 API 密钥、密码等）
- 建议为重要文件添加注释说明其用途
